# ⏱️ Timeout 倒计时自动锁屏

一个极简的 Android 定时锁屏工具，支持自定义分钟倒计时，时间到自动锁屏，告别拖延。

---

## ✨ 功能特性
- 🔢 **自定义倒计时**：默认 30 分钟，可任意修改时长
- ⏰ **分秒实时显示**：`MM:SS` 格式倒计时，直观清晰
- 🎨 **极简蓝色主题**：清爽界面，和图标风格统一
- 🔒 **自动锁屏**：倒计时结束，设备自动锁定
- 💪 **后台稳定运行**：华为手机也能稳定保活，不被系统杀掉
- 🚫 **防重复点击**：倒计时期间按钮禁用，避免误操作
- 🎛️ **立体弹窗输入**：带阴影圆角的输入框，质感高级

---

## 📱 使用说明
1.  **开启设备管理员权限**
    首次打开 APP 会请求设备管理员权限，这是自动锁屏功能的必要前提，请选择「允许」。

2.  **设置倒计时**
    点击「开始倒计时」，在弹窗中输入分钟数（默认 30），点击「开始」即可启动。

3.  **等待倒计时结束**
    主界面会实时显示剩余时间，倒计时结束后设备会自动锁屏。

## ⚠️ 注意事项
- 自动锁屏需要**设备管理员权限**，卸载 APP 前请先在设置中取消该权限。
- 后台保活效果与系统设置有关，建议在华为手机中给 APP 开启「允许后台活动」权限。
- 项目已移除截屏功能，仅保留基础倒计时与锁屏逻辑，稳定不闪退。
